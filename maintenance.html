系統維護

可用這個頁面新增、修改、刪除入口卡片資料，完成後下載新的 platforms.json。這是免費靜態網站的安全維護方式，不會把 GitHub 密碼或權杖放在網頁裡。

操作方式：先在左側選平台，在右側修改資料；若要新增平台，按「新增空白平台」。完成後按「下載 platforms.json」，再把下載的檔案上傳覆蓋 GitHub repo 裡的 data/platforms.json。
